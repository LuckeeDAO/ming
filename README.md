## 📋 项目简介

Ming（明命）是一个基于东方命理哲学与集体意识共识的 Web3 仪式平台。在天地人的框架中，人类共识是我们可以共同构建、共同引导的能量。命不仅包括自身的命理，还包括各种自然规律（如节气、星宿、五行循环）和社会规律（如共识、集体意识）。因此，Ming 通过可编程的仪式 NFT 与链上契约，将个人命理调理融入群体共同参与的仪式场中，帮助用户在顺应天道（自然与社会规律）的基础上，以共识为桥，以仪式为媒，达成生命的疏通与升华。

我们不止于预测，更致力于构建一种人与命运共修的新范式——在这里，每一次 mint、每一次祈愿、每一次共识的达成，都是对自我命途的一次温和修正与能量加冕。

### 🎯 平台内核

基于四柱八字五行生克理论，以 NFT 为能量见证载体，融合链上可验证仪式与集体意识共识机制，构建**"命理－共识－仪式"三元闭环**。

命不仅包括个人命理，还包括自然规律（节气、星宿、五行）与社会规律（共识、集体意识），通过三元闭环实现个人与自然、社会的和谐共振。

### 💡 核心主张

- **天命不可违，但人道可通**：在顺应天道（自然规律与社会规律）的基础上，通过人道的力量与共识的能量，达成生命的疏通与升华。我们尊重天命轨迹（包括自然规律与社会规律），同时相信人道有其力量
- **共识即能量，仪式即通道**：人类共识是我们可以共同构建、共同引导的能量，是社会规律的重要组成部分。通过链上仪式，将个人命理调理融入群体共同参与的仪式场中，形成个人与自然规律、社会规律的共振与循环
- **以人合天，以链载愿**：通过 Web3 技术与区块链共识，将传统命理智慧与现代技术结合，以可编程的仪式 NFT 承载祈愿，实现个人与自然规律、社会规律的和谐共续

### 🌟 体验形态

用户通过完成对应五行、节气、星宿的链上仪式，铸造专属能量 NFT，同时接入相应共识池，形成个人与集体能量场的共振与循环。

### 🔄 核心思想

- **命（先天框架）**: 命不仅包括个人命理（四柱八字），还包括自然规律（节气、星宿、五行循环）与社会规律（共识、集体意识），这些构成了不可修改的先天框架
- **明（知命）**: 通过分析了解自己的能量系统，同时认知自然规律与社会规律，明白命的全貌
- **协同调节（借物调运）**: 通过外物连接，借助公网共识见证，在顺应自然与社会规律的基础上，优化个人选择与行动节律
- **共识成仪**: 通过集体意识共识机制，形成个人与自然规律、社会规律的共振与循环

## 🚀 快速开始

### 环境要求

- **Node.js**: 18.0+ (推荐 20.x)
- **npm**: 9.0+ 或 **yarn**: 1.22+
- **Git**: 2.30+

### 安装和启动

```bash
# 克隆项目
git clone git@github.com:LuckeeDAO/ming.git
cd ming

# 进入前端目录并安装依赖
cd srcs
npm install

# 启动开发服务器
npm run dev

# 访问应用
open http://localhost:3000
```

### 开发命令

```bash
# 开发环境
npm run dev                    # 启动开发服务器

# 构建项目
npm run build                  # 生产构建
npm run preview                # 预览构建结果

# 代码质量
npm run lint                   # 代码检查
npm run format                 # 代码格式化

# 测试
npm run test                   # 单元测试
npm run test:ci                # CI模式测试（无交互）
npm run test:ui                # 测试UI
npm run test:e2e               # 端到端测试
```

Solana 合约（Anchor）验证：

```bash
cd contracts/solana
npm install
anchor build
anchor test
```

说明：`anchor build/test` 需要可访问 crates.io 与 Solana 依赖源的网络环境。

## 🌐 功能特性

### 核心功能

## 🔐 安全与隐私

### 数据安全

- **链上存储**: NFT所有权、连接记录、能量场见证
- **IPFS存储**: 图片、元数据、详细记录
- **本地存储**: 用户临时数据、未发布内容

### 隐私保护

- 四柱八字信息不上链（仅分析结果上链）
- 用户可选择感受记录是否公开
- 本地数据加密存储

## 📈 三阶段发展路线图

> 详细的三阶段发展路线图请参考：[三阶段发展路线图](./docs/02-概要设计/05-三阶段发展路线图.md)

### 第一阶段：共识铸造厂（Mint of Consensus）

**口号**：你的信念，有迹可循。

**核心目标**：验证核心假设，建立品牌心智，积累用户基础

**核心功能**:
- ✅ 五行能量徽章NFT（补木/火/土/金/水）
- ✅ 节气节日仪式NFT（限时铸造）
- ✅ 祝福传递NFT（自定义祝福语）
- ✅ 共识池展示（实时参与人数）
- ✅ 持有者图谱（全球分布）
- ✅ 铸造仪式感（动画交互）

**技术栈**:
- 标准ERC-721/ERC-1155合约
- 链上事件驱动的共识池
- IPFS存储
- 基础用户仪表盘

### 第二阶段：命理调节舱（Chamber of Destiny Adjustment）

**口号**：共识导航你与生俱来的能量地图。

**核心目标**：构建核心壁垒，提供个性化深度价值

**核心功能**:
- ✅ 个人命盘引擎（四柱八字分析、十神、格局）
- ✅ 个性化NFT推荐系统（基于命盘喜用神）
- ✅ 定制化仪式NFT（年度调理计划）
- ✅ 能量仪表盘（五行可视化、NFT贡献度）
- ✅ 周期仪式订阅（基于流年提醒）
- ✅ 组合效应系统（NFT组合达成能量目标）

**技术栈**:
- 命理算法引擎（chinese-lunar-master集成）
- 推荐算法系统
- 动态元数据生成
- 隐私保护（本地计算）

### 第三阶段：灵境绘影师（AI Iconographer of the Soul）

**口号**：让你看不见的能量，拥有独一无二的容颜。

**核心目标**：提升体验与生态，形成完整价值闭环

**核心功能**:
- ✅ AI能量肖像生成（基于命盘和共识NFT）
- ✅ 动态能量视觉化（时间、共识池、用户行为）
- ✅ 图鉴与叙事（能量构成解读、AI故事）
- ✅ 社区共创模型（东方美学风格训练）
- ✅ UGC工具和API（社区二次创作）

**技术栈**:
- AI图像生成模型（Stable Diffusion定制化）
- Prompt引擎（命理/共识数据 → 图像提示词）
- 动态元数据更新
- 社区数据收集机制
