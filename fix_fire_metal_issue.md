# 修复方案：火比金弱的问题

## 问题确认

对于八字"乙卯丁亥辛未壬辰"（日主：辛金），在月令校正后：
- 火能量：811.2
- 金能量：835.2
- **火比金弱**，这不符合常理

## 根本原因

### 1. 月令校正系数过于极端
- 亥月（冬季）对火的校正系数是**0.52**（非常弱）
- 亥月对金的校正系数是**0.72**（也弱，但比火强38%）
- 虽然冬季火弱是合理的，但0.52可能过于极端

### 2. 水多克火，雪上加霜
- 水能量（2332.8）远大于火能量（811.2），比例约2.87:1
- 水克火的相克关系会进一步削弱火能量
- 而火克金虽然也存在，但火本身已经很弱，对金的克制作用有限

### 3. 相生关系可能无法弥补
- 木多（3540）应该能生火，但：
  - 木生火的效率受多种因素影响（阴阳、位置、能量比等）
  - 水克火的力量可能抵消了木生火的效果

## 修复建议

### 方案1：调整月令校正系数（推荐）

检查并调整亥月对火的校正系数，使其更合理：

```typescript
// 当前代码：energyQuantitativeService.ts 第183行
亥: { water: 1.08, wood: 1.0, fire: 0.52, earth: 0.78, metal: 0.72 },

// 建议调整为：
亥: { water: 1.08, wood: 1.0, fire: 0.60, earth: 0.78, metal: 0.72 },
// 或者
亥: { water: 1.08, wood: 1.0, fire: 0.65, earth: 0.78, metal: 0.72 },
```

**理由**：
- 虽然冬季火弱，但0.52可能过低
- 调整为0.60-0.65，既能体现冬季火弱，又不会过于极端
- 这样火能量会从811.2提升到约936-1054，可能超过金能量

### 方案2：检查相生相克的平衡

确保相生关系（木生火）能够有效发挥作用，不被相克关系完全抵消。

### 方案3：考虑日主特殊性

虽然日主是辛金，但日主的能量应该通过其他机制（如得根得气、位置权重等）来体现，而不是简单地让金能量最高。

## 验证步骤

1. 修改月令校正系数
2. 重新计算八字能量
3. 验证火能量是否合理
4. 对比其他类似八字，确保修改不会影响其他情况

## 注意事项

- 修改月令校正系数会影响所有亥月的八字计算
- 需要确保修改后的系数在整体算法中仍然合理
- 建议先在小范围内测试，再全面应用
