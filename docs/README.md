## 📖 文档说明

### 01-项目概述
包含项目背景、核心理念和项目定位等基础信息。

### 02-概要设计
系统整体架构设计、技术选型、功能模块和数据模型的高层设计。

### 03-详细设计
各模块的详细设计文档，包括前端、智能合约、API、数据库和UI/UX设计。

### 04-开发指南
开发环境搭建、代码规范、项目结构和测试指南。

### 05-部署运维
部署方案、运维监控和安全规范。

### 06-其他
术语表、参考资料和项目更新日志。
