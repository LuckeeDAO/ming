# 核心理念

## 1. 平台核心理念

### 1.1 理念表述

**"缺失认知，外物连接，NFT见证，仪式完成"**

### 1.2 理念解析

#### 缺失认知
- 通过四柱八字分析，识别能量系统中的缺失元素
- 提供定性的认知框架，而非精确的量化分析
- 帮助用户了解自己的能量状态

#### 外物连接
- 根据能量缺失，推荐合适的外物进行连接
- 外物包括自然物、矿物、植物等
- 通过象征性连接补充缺失的能量

#### NFT见证
- 使用NFT作为能量场见证的凭证
- 借助公网（区块链）的共识作为能量场
- 记录连接仪式的重要时刻

#### 仪式完成
- 完整的仪式流程和体验
- 强调过程的价值，而非结果的验证
- 为个人成长创造有意义的时刻

## 2. 哲学思想

### 2.1 命与明

**命（先天框架）**:
- 四柱八字是先天给定的命运框架
- 这个框架是不能修改的
- 是"命"的体现

**明（知命）**:
- 通过分析了解自己的能量系统
- 认识到自己的能量状态和缺失
- 是"明"的体现

### 2.2 改命思想

**核心思想**: 上天给了一个命运的安排，而且是不能修改的，那么我们后天就要强行向天地借物改命。

**实现方式**:
1. 分析自己的能量系统（知命）
2. 识别能量循环的缺失
3. 通过外物连接补充缺失
4. 借助公网共识作为能量场见证
5. 完成改命仪式

## 3. 理论基础

### 3.1 四柱八字

四柱八字是传统命理学的核心理论：

- **年柱**: 出生年份的天干地支
- **月柱**: 出生月份的天干地支
- **日柱**: 出生日期的天干地支
- **时柱**: 出生时辰的天干地支

通过四柱八字可以分析：
- 五行能量分布
- 能量循环状态
- 能量缺失情况

### 3.2 五行能量

五行包括：
- **木**: 生长、创造、活力
- **火**: 热情、行动、光明
- **土**: 稳定、安全、承载
- **金**: 规则、决断、收敛
- **水**: 智慧、灵活、流动

### 3.3 能量循环

能量需要在系统中循环流通：
- **流通顺畅**: 能量能够正常循环
- **流通阻塞**: 能量循环受阻
- **能量缺失**: 某种能量类型不足

### 3.4 外物连接

通过自然外物补充缺失的能量：
- **木元素**: 森林、树木、植物
- **火元素**: 太阳、火焰、光明
- **土元素**: 大地、山脉、陶土
- **金元素**: 矿石、金属、高山
- **水元素**: 海洋、河流、雨水

## 5. 设计原则

### 5.1 极简主义

- **信息极简**: 每页核心信息不超过3个
- **操作极简**: 核心操作不超过3步
- **视觉极简**: 清晰的信息层次，适度的留白

### 5.2 仪式感优先

- **过程仪式感**: 每个步骤有明确的开始和结束
- **视觉仪式感**: 重要的视觉提示和完成效果
- **体验仪式感**: 音效和动画的适度使用
