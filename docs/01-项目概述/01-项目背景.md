# 项目背景

## 1. 项目起源

### 1.1 核心理念

Ming（命/明）平台源于对传统命理学的现代思考：

- **命（先天框架）**: 四柱八字是先天给定的命运框架，是"命"的体现
- **明（知命）**: 通过分析了解自己的能量系统，是"明"的体现
- **改命（借物改命）**: 通过外物连接，借助公网共识能量场，强行改变命理布局

### 1.2 哲学思想

**核心哲学**: 上天给了一个命运的安排，而且是不能修改的，那么我们后天就要强行向天地借物改命。

### 1.3 理论基础

- **四柱八字**: 传统命理学的核心理论
- **五行能量**: 木、火、土、金、水五种能量类型
- **能量循环**: 能量需要在系统中循环流通
- **外物连接**: 通过自然外物补充缺失的能量

## 2. 项目定位

### 2.1 平台定位

Ming平台是一个**外物连接NFT仪式平台**，专注于：

- 提供能量循环缺失的定性认知框架
- 指导用户通过NFT仪式与自然外物建立象征性连接
- 重点在于连接仪式本身，而非效果量化验证
- 提供可执行、可完成的实际仪式流程

### 2.2 目标用户

- 对传统文化感兴趣的用户
- 寻求个人成长和仪式体验的用户
- 对Web3和NFT技术感兴趣的用户
- 希望探索命理学现代应用的用户

### 2.3 平台价值

- **仪式感价值**: 提供有意义的象征性连接仪式
- **过程体验价值**: 完整的连接过程体验
- **文化表达价值**: 传统文化元素的现代表达
- **个人意义价值**: 为个人成长创造有意义的时刻

## 3. 项目目标

### 3.1 短期目标（MVP）

- 实现基础的四柱八字分析功能
- 实现外物推荐和选择功能
- 实现NFT铸造功能
- 实现个人档案管理功能

### 3.2 中期目标

- 完善仪式指南和素材库
- 实现多语言支持
- 优化用户体验
- 建立用户社群

### 3.3 长期愿景

- 成为传统文化现代表达的平台
- 建立有意义的数字仪式社群
- 探索文化传承的创新模式
- 创造有社会价值的文化产品

## 4. 项目特色

### 4.1 技术特色

- **Web3技术**: 使用区块链和NFT技术
- **去中心化**: 使用公网共识作为能量场见证
- **现代架构**: 基于React和TypeScript的现代前端

### 4.2 文化特色

- **传统文化**: 基于四柱八字和五行理论
- **现代表达**: 用现代技术表达传统文化
- **仪式感**: 强调仪式过程的体验价值

### 4.3 设计特色

- **极简主义**: 信息极简、操作极简、视觉极简
- **仪式感**: 营造庄重、有意义的仪式体验
- **用户友好**: 清晰直观，易于理解和使用
