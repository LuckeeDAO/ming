# 钱包接口文档

> 基线版本：v1.0（参照 `docs/00-现行口径基线.md`）

本目录用于外部钱包工程对接 Ming 的钱包能力。

## 文档分工（单一权威）

1. `钱包接口技术规范.md`：唯一技术权威文档（协议、字段、流程、错误码、安全约束、联调要求）
2. `权限路径决策与实施清单.md`：铸造权限模型（多类用户可铸造）与风控实施项
3. `钱包联调命令清单.md`：可直接执行的联调命令与预期结果
4. `钱包联调验收清单.md`：联调与上线前验收清单（执行与留痕）
5. `AnDaoWallet联调专项清单.md`：AnDaoWallet 工程专项联调样例与验收项
6. `钱包NFT铸造功能需求说明书.md`：背景与决策记录（非技术权威）

## 使用顺序

1. 先阅读 `钱包接口技术规范.md`
2. 确认 `权限路径决策与实施清单.md` 中权限模型与控制项已满足
3. 按 `钱包联调命令清单.md` 执行基础联调与验证
4. 若对接 AnDaoWallet，追加执行 `AnDaoWallet联调专项清单.md`
5. 再按 `钱包联调验收清单.md` 执行验收与留痕
6. 仅在需要追溯背景时查阅 `钱包NFT铸造功能需求说明书.md`
